## Projetos

### Atlas Farmaco
Um catálogo interativo de psicofármacos voltado para profissionais de saúde, estudantes e curiosos. Reúne informações sobre mecanismo de ação, indicações clínicas, usos off-label, efeitos adversos e interações farmacológicas dos principais medicamentos utilizados na psiquiatria brasileira.

- Stack: React, TypeScript, Vite, Google AI Studio (Gemini)
- Deploy: [atlas-farmaco.vercel.app](https://atlas-farmaco.vercel.app)
- Repositório: [github.com/thomashnblum/atlas-farmaco](https://github.com/thomashnblum/atlas-farmaco)

### Philosophy Quiz
Um quiz interativo sobre filosofia, integrado à API do Gemini para geração dinâmica de perguntas e respostas. Foi o meu primeiro projeto web completo, do zero ao deploy.

- Stack: TypeScript, Vite, Node.js, Gemini API
- Deploy: GitHub Pages
- Repositório: [github.com/thomashnblum/philosophy_quiz](https://github.com/thomashnblum/philosophy_quiz)
